= Moria View Pack - A better UI/UX for Redmine

Redmine is a great software to manage projects of nearly all kind. Besides its superior functionality it lacks heavily in terms user friendlieness. Moria is the attempt to fix this problem and to make Redmine as user friendly as possible while relying on Redmines base views and styles. Moria is developed with small screens in mind so it should provide a great experience on mobile devices, too.
